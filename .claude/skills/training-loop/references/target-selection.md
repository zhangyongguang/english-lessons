# Target selection

Use the requested week plus the preceding two ISO weeks.

Rank candidates using:

1. Frequency in the requested week.
2. Number of distinct days on which the problem appeared.
3. Recurrence across preceding weeks.
4. Effect on intelligibility and sentence construction.
5. Value of automating a high-frequency foundational form.
6. Existing mastery state and scheduled review needs.

Prefer a foundational recurring problem over an isolated advanced expression. Treat apparent transcription noise cautiously. Use teacher-supported corrections where possible.

Each target must describe one behavior that Chat Live can observe repeatedly, such as “use past-tense verbs when recounting completed events.” Do not choose a raw tag when it contains several unrelated behaviors.

Select exactly three targets. Cite recent-error counts, distinct days, and two or three representative examples. Do not expose dozens of source errors in the training prompt.
